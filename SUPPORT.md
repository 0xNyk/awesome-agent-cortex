# Support

## Resource questions

Use [GitHub Discussions](https://github.com/0xNyk/awesome-agent-cortex/discussions) for questions about the directory, category boundaries, and broader curation proposals.

## Bugs and stale resources

Use the structured [issue forms](https://github.com/0xNyk/awesome-agent-cortex/issues/new/choose) for broken links, outdated descriptions, or new resource proposals.

Include the affected section, canonical URL, and evidence from the project owner or primary documentation. Maintainer support is best-effort; inclusion does not transfer support responsibility from a listed project to this repository.

## Security

Follow [SECURITY.md](SECURITY.md) for private vulnerability reporting. Do not publish unpatched security details in an issue or discussion.
